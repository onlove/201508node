#一个普通网站的访问流程
1.浏览器（其它的客户端）向服务器发送一个请求
2.先解析域名,DNS域名解析服务器(chrome://net-internals/#dns->搜索操作系统缓存->读取本地host文件->DNS服务器)
3.客户端通过随机端口通过HTTP三次握手，与服务器建立连接
4.浏览器要发送http请求
5.服务器接收到浏览器的请求，解析请求参数和路径，经过后台处理返回数数
